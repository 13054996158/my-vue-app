# my-vue-app

## 项目介绍
这是一个基于 Vue 3 和 Element Plus 的多标签页示例项目，包含路由与标签页同步管理功能，支持标签页切换、关闭和动态添加，适合用作后台管理系统基础框架。

## 功能特点
- 多标签页（Tab）管理
- 路由与标签页同步，保证页面切换和标签高亮一致
- 支持关闭标签页，自动切换到上一个标签页
- 通过 `router-view` 的 `key` 强制刷新路由组件
- 简单易扩展，适合二次开发

## 技术栈
- Vue 3
- Vue Router 4
- Element Plus
